### Phase 3: The 4 Stages Logic
**1. What Do You Know?:**
- Turn-based. A player has exactly a 6-second timer to type when it's their turn.
- Timeout (6s ends) = "Strike", and the turn moves to the next player automatically.

**2. The Auction (Bidding):**
- Bidding phase: 6-second timer to place a bid. Timeout = player is skipped for this question.
- Answering phase: Winning bidder types answers against a descending clock. 
- UI shows real-time progress to everyone (e.g., "1/8 answers"). Wrong answers show as "1 mistake" (no limit). Right answers increment the counter.

**3. The Buzzer Round (Speed):**
- First to click the on-screen buzzer gets the chance to type.
- *Tech Constraint:* Use Server-Side Timestamps to determine who pressed the buzzer first to avoid latency disputes.
- Correct = point. Wrong = locked out, and others can buzz in. 

**4. Guess The Player (Simultaneous):**
- Player career path appears. ALL players can type simultaneously.
- Wrong guesses are broadcasted to all screens. First player to type the correct answer wins the point.

- **Tie-Breaker:** If tied, trigger a single "Sudden Death" round (Guess the Player, Medium). First correct answer wins. Add a "Change Player" button (requires unanimous vote to skip).
- **Graceful Reconnect:** If a player's internet drops, give them a 30-second window to reconnect to the exact same room/state without getting kicked.
